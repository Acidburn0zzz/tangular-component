## j-FontAwesomeBox

A simple __FontAwesome__ viewer with fulltext searching.

__Configuration__:

- `search` {String} a placeholder for the search input
- `height` {Number} Optional, default: `250`
- `fa` {Boolean} Optional, `true` leaves `fa-` prefix (default: `false`)
