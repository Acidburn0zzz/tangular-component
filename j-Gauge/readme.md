## j-Gauge

__Configuration__:

- `decimals` {Number} a count of decimals (default: `1`)
- `format` {String} a format for text (default: `{0}%`)
- `colors` {String} colors (default: `30 #8CC152,40 #EDBC5A,30 #DB3737`)
- `text` {Boolean} enables text (default: `true`)
