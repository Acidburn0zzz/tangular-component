## j-Windows (BETA)

- Works only with `+v18`
- Supports mobile devices
- Supports touches
- Movable
- Resizable

__Configuration__:

- `menuicon` {String} a menu icon in window title bar (default `fa fa-navicon`)

The `windows` component expects `Array` of objects.
